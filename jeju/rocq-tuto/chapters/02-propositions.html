Propositions as Types

In Rocq, there's a beautiful idea: propositions are types, and proofs are programs. This is called the Curry-Howard correspondence.

  • A proposition P is a type
  • A proof of P is a term of type P
  • An implication A -> B is a function type
curry_howard.v 
(* True has exactly one proof: I *)
Check I.       (* I : True *)

(* An implication A -> B is like a function type *)
Check (fun (H : True) => H).   (* True -> True *)

(* A proof of A -> B is a function that transforms
   evidence of A into evidence of B *)

The Core Tactics

intro / intros

When your goal is A -> B, intro H moves A into your hypotheses as H.

Before intro H
A -> B

⬇️ intro H.

After intro H
H : A
B

exact

When your goal exactly matches a hypothesis, use exact H.

apply

If you have H : A -> B and your goal is B, then apply H changes the goal to A.

assumption

Like exact, but Rocq finds the matching hypothesis automatically.

Examples

examples.v 
(* Identity: proving A -> A *)
Theorem identity : forall A : Prop, A -> A.
Proof.
  intro A.      (* Introduce the type A *)
  intro H.      (* Introduce hypothesis H : A *)
  exact H.      (* Goal is A, we have H : A *)
Qed.

(* Composition of implications *)
Theorem impl_trans : forall A B C : Prop,
  (A -> B) -> (B -> C) -> (A -> C).
Proof.
  intros A B C HAB HBC HA.
  apply HBC.    (* Now we need B *)
  apply HAB.    (* Now we need A *)
  exact HA.
Qed.

(* The K combinator *)
Theorem K : forall A B : Prop, A -> B -> A.
Proof.
  intros A B HA HB.
  exact HA.
Qed.

(* The S combinator *)
Theorem S : forall A B C : Prop,
  (A -> B -> C) -> (A -> B) -> A -> C.
Proof.
  intros A B C HABC HAB HA.
  apply HABC.
  - exact HA.
  - apply HAB. exact HA.
Qed.

More Tactics

auto

Automatically searches for a proof using introduction rules and hypotheses.

assert

Prove a helper lemma in the middle of your proof.

auto_assert.v 
(* auto can solve many simple goals *)
Theorem auto_example : forall A B C : Prop,
  A -> (A -> B) -> (B -> C) -> C.
Proof.
  auto.
Qed.

(* Using assert for intermediate lemmas *)
Theorem using_assert : forall A B C : Prop,
  (A -> B) -> (B -> C) -> A -> C.
Proof.
  intros A B C HAB HBC HA.
  assert (HB : B).
  { apply HAB. exact HA. }
  apply HBC. exact HB.
Qed.

Summary

  • Propositions are types, proofs are terms
  • intro / intros — introduce hypotheses
  • exact — use an exact matching hypothesis
  • apply — use an implication backward
  • assumption — let Rocq find the hypothesis
  • auto — automatic proof search
  • assert — prove intermediate lemmas
